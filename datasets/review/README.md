# Processing Artifacts (No-Review Default)

By default, the pipeline does not require human review. This directory stores
intermediate artifacts used for tests, regression checks, and traceability.

Main files:

- `raw_name_evidence_v1.jsonl`
- `normalized_names_v1.jsonl`
- `run_metadata_v1.json`

Legacy review files may still appear if candidate-generation commands are used,
but they are optional and not required for the primary lexicon build.
